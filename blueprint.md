
# Lotto Number Generator

## Overview

This is a simple web application that generates random lottery numbers. Users can click a button to generate a new set of numbers and view a history of previously generated numbers.

## Current Plan

### Feature: Lotto Number Generation

*   **Goal:** Create a user interface to generate and display lottery numbers.
*   **Steps:**
    1.  Update the title and heading of the `index.html` page.
    2.  Create a designated area to display the generated lottery numbers.
    3.  Add a button that, when clicked, will trigger the number generation.
    4.  Implement the JavaScript logic in `main.js` to generate 6 unique random numbers between 1 and 45.
    5.  Display the generated numbers in the designated area.
    6.  Add styling in `style.css` to create a visually appealing layout for the number display and button.
